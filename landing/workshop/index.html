1 Install OpenClaw

Option A: Docker (Recommended)

# Pull and run
docker run -it ghcr.io/anthropics/openclaw:latest

Option B: npm

# Install globally
npm install -g openclaw

# Or use npx
npx openclaw

Option C: From Source

git clone https://github.com/openclaw/openclaw
cd openclaw
npm install
npm start

2 Configure Your Agent

Run the setup wizard:

openclaw configure
You'll need:
  • üîë AI API Key ‚Äî Anthropic (Claude), OpenAI, or Groq
  • üì± Messaging ‚Äî Telegram bot token (optional)

Get a Free API Key

Groq (Free tier): console.groq.com

Anthropic: console.anthropic.com

3 Connect Telegram

Create a Telegram Bot

  1. Open Telegram, search for @BotFather
  2. Send /newbot
  3. Choose a name (e.g., "My OpenClaw Agent")
  4. Choose a username (e.g., "my_openclaw_bot")
  5. Copy the API token
# Add to your config
openclaw configure --section telegram

# Or set environment variable
export TELEGRAM_BOT_TOKEN="your-token-here"

4 Run Your Agent

# Start the gateway
openclaw gateway start

# Check status
openclaw status
